## 1. Camera Movement

### Basic Movements

| Movement | Description | Effect | Prompt Keywords |
|----------|-------------|--------|-----------------|
| **Static** | Camera locked, no movement | Stable, observational | "static shot", "locked camera" |
| **Pan** | Rotate left/right on axis | Survey, follow action | "pan left/right", "horizontal pan" |
| **Tilt** | Rotate up/down on axis | Reveal height, scale | "tilt up/down", "vertical tilt" |
| **Dolly** | Move toward/away from subject | Intimacy/distance | "dolly in/out", "push in" |
| **Truck/Crab** | Move left/right alongside | Parallel tracking | "truck left/right", "crab" |
| **Pedestal** | Move camera up/down vertically | Reveal, follow | "pedestal up/down" |

### Advanced Movements

| Movement | Description | Effect | Prompt Keywords |
|----------|-------------|--------|-----------------|
| **Tracking** | Follow subject movement | Immersion, action | "tracking shot", "follow shot" |
| **Crane/Jib** | Rise/lower on crane arm | Epic, reveal, establish | "crane shot", "jib up/down" |
| **Steadicam** | Smooth handheld with rig | Fluidity, exploration | "steadicam", "smooth follow" |
| **Handheld** | Camera operator holds | Realism, tension, energy | "handheld", "documentary style" |
| **Gimbal** | Stabilized handheld | Modern smooth movement | "gimbal shot", "stabilized" |

### Signature Movements

| Movement | Description | Famous Use | Prompt Keywords |
|----------|-------------|------------|-----------------|
| **Dolly Zoom (Vertigo)** | Dolly + zoom opposite | Vertigo, Jaws | "dolly zoom", "vertigo effect" |
| **Whip Pan** | Very fast pan | Transitions, energy | "whip pan", "snap pan" |
| **360° Orbit** | Circle around subject | Dramatic, hero | "orbital shot", "360 around" |
| **Oner/Long Take** | Extended single shot | Immersion | "long take", "single shot" |

### Movement Speed & Feel

| Speed | Effect | Prompt Keywords |
|-------|--------|-----------------|
| **Very slow** | Contemplative, dreamlike | "slow motion", "languid" |
| **Slow** | Elegant, observational | "slow dolly", "gentle push" |
| **Medium** | Natural, conversational | "steady", "smooth" |
| **Fast** | Energy, urgency | "quick pan", "rapid" |
| **Whip** | Jarring, transition | "whip", "snap" |

## 2. Shot Types

### By Distance (Frame Size)

| Shot | Frame | Purpose | Prompt Keywords |
|------|-------|---------|-----------------|
| **Extreme Wide (EWS)** | Vast landscape, tiny subject | Establish scale, isolation | "extreme wide shot", "vast landscape" |
| **Wide/Establishing** | Full location | Set scene, context | "wide shot", "establishing shot" |
| **Full Shot** | Whole body, head to toe | Action, body language | "full shot", "full body" |
| **Medium Wide (MWS)** | Knees up | Action + expression | "medium wide", "cowboy shot" |
| **Medium (MS)** | Waist up | Conversation | "medium shot", "waist up" |
| **Medium Close-up (MCU)** | Chest up | Emotion + context | "medium close-up", "chest up" |
| **Close-up (CU)** | Face fills frame | Emotion, intimacy | "close-up", "face" |
| **Extreme Close-up (ECU)** | Detail (eyes, hands) | Intensity, detail | "extreme close-up", "detail shot" |

### By Angle

| Angle | Camera Position | Effect | Prompt Keywords |
|-------|-----------------|--------|-----------------|
| **Eye Level** | Subject's eye height | Neutral, relatable | "eye level" |
| **Low Angle** | Below, looking up | Power, heroic | "low angle", "looking up" |
| **High Angle** | Above, looking down | Vulnerable, weak | "high angle", "looking down" |
| **Dutch/Canted** | Tilted horizon | Unease, tension | "dutch angle", "canted" |
| **Bird's Eye** | Directly above | Overview, god-like | "bird's eye", "overhead" |
| **Worm's Eye** | Ground level up | Imposing, dramatic | "worm's eye", "ground level" |

### By Subject Relationship

| Shot | Description | Use | Prompt Keywords |
|------|-------------|-----|-----------------|
| **POV** | Character's viewpoint | Immersion, tension | "POV shot", "point of view" |
| **Over-the-shoulder (OTS)** | Behind one person | Dialogue, connection | "over the shoulder", "OTS" |
| **Two-shot** | Two people in frame | Relationship | "two-shot" |
| **Reaction shot** | Character reacting | Emotion, emphasis | "reaction shot" |
| **Insert/Cutaway** | Detail, object | Information, transition | "insert shot", "cutaway" |

## 3. Shot Sequences

### Classic Editing Pattern

```
Establishing (Wide) → Medium → Close-up → Reaction
```

### Scene Coverage

| Purpose | Shot Sequence |
|---------|---------------|
| **New location** | EWS → WS → MS (characters enter) |
| **Dialogue** | OTS A → OTS B → Reaction → Two-shot |
| **Action** | WS (context) → MS (action) → CU (impact) |
| **Reveal** | CU (detail) → Dolly out → WS (reveal) |
| **Tension build** | WS → MS → CU → ECU (escalating) |

### Transition Techniques

| Technique | Description | Prompt Keywords |
|-----------|-------------|-----------------|
| **Cut** | Direct change | (default) |
| **Dissolve** | Blend between shots | "dissolve to", "cross-fade" |
| **Fade** | To/from black | "fade to black", "fade in" |
| **Wipe** | Edge moves across | "wipe transition" |
| **Match cut** | Similar shapes/motion | "match cut on..." |

## 4. Aspect Ratios

### Common Ratios

| Ratio | Name | Feel | Use |
|-------|------|------|-----|
| **1.33:1 (4:3)** | Academy | Vintage, intimate, TV | Classic films, period pieces |
| **1.78:1 (16:9)** | HD/Widescreen | Modern, balanced | TV, YouTube, streaming |
| **1.85:1** | Academy Flat | Theatrical, subtle wide | Most modern films |
| **2.00:1** | Univisium | Compromise | Netflix originals |
| **2.35:1 / 2.39:1** | Scope/Anamorphic | Epic, cinematic | Blockbusters, epics |
| **2.76:1** | Ultra Panavision | Ultra-epic | Hateful Eight |
| **1:1** | Square | Balanced, social | Instagram, artistic |
| **9:16** | Vertical | Mobile-first | TikTok, Reels, Stories |

### Anamorphic Look

Characteristics:
- Horizontal lens flares
- Oval/elliptical bokeh
- Edge distortion (barrel)
- Distinctive "squeezed" feel

**Prompt keywords:** "anamorphic widescreen", "oval bokeh", "horizontal lens flare", "cinemascope"

### Choosing Ratio

| Content | Recommended |
|---------|-------------|
| Epic landscapes, action | 2.35:1 |
| Character drama | 1.85:1 or 16:9 |
| Intimate, personal | 4:3 |
| Social media | 16:9 (YT), 9:16 (TikTok), 1:1 (IG) |
| Vintage feel | 4:3 with grain |

## 5. Pacing & Rhythm

### Shot Duration

| Duration | Feel | Use |
|----------|------|-----|
| **< 2 sec** | Fast, energetic | Action, montage |
| **2-4 sec** | Normal, conversational | Dialogue, general |
| **4-8 sec** | Measured, contemplative | Drama, beauty shots |
| **8+ sec** | Slow, meditative | Art film, tension |

### Cutting Rhythm

| Rhythm | Pattern | Effect |
|--------|---------|--------|
| **Staccato** | Quick, short cuts | Energy, chaos |
| **Legato** | Long, flowing | Calm, dreamlike |
| **Accelerating** | Cuts get faster | Building tension |
| **Decelerating** | Cuts slow down | Resolution, calm |

### Scene Pacing

| Scene Type | Pacing Strategy |
|------------|-----------------|
| **Action** | Fast cuts, dynamic movement |
| **Dialogue** | Medium pacing, reaction shots |
| **Romance** | Slow, lingering, close-ups |
| **Horror** | Slow build → sudden cuts |
| **Comedy** | Timing-dependent, reaction holds |

## Video Prompt Checklist

```
□ Shot type (wide, medium, close-up)
□ Camera movement (static, pan, dolly, tracking)
□ Movement speed (slow, medium, fast)
□ Subject action (what are they doing)
□ Direction (left to right, toward camera)
□ Duration (how long)
□ Aspect ratio (16:9, 2.35:1, etc.)
□ Atmosphere (lighting, color, mood)
□ Style reference (cinematic, documentary, etc.)
```
